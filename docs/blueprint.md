# **App Name**: RetailPass

## Core Features:

- User Authentication: User authentication: securely manages user logins, providing registered user login and new user signup features
- Profile Management: Profile Management: Allows users to update their account details, password.
- Session Management: Session Handling: Manages user sessions for persistent logins.
- Password Strength Analyzer: Password Strength Tool: Analyzes entered password to assess if it meets requirements

## Style Guidelines:

- Primary color: Muted Blue (#6495ED) for trust and reliability.
- Background color: Light Gray (#E6E6E6) to create a clean and accessible backdrop.
- Accent color: Soft Purple (#A084CA) for interactive elements, drawing the user's attention to actionable components.
- Body and headline font: 'Inter' (sans-serif) for clear and modern text.
- Use minimalist icons for account and security settings.
- Clean and centered layout to improve readability and focus on the login/profile forms.
- Subtle transitions and loading animations to enhance user experience.
